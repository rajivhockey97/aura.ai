🔑 How to Get a Free Gemini API Key

  1. Go to the official Google AI Studio console.
  2. Log in with any personal Google/Gmail account.
  3. Click the blue "Get API key" button in the left sidebar.
  4. Click "Create API key", then select an existing Google Cloud project or create a new one.
  5. Copy the generated API key (it starts with AIzaSy...).
  6. Paste it into the API input box above, then click "Save Configurations".

Note: Standard accounts get free, rate-limited access which is more than enough for playing and testing this quizzing application.
